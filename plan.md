# 项目改造计划：AI 聊天助手

本项目旨在将现有的 Vue 应用改造为一个功能完善的、与大语言模型（LLM）进行实时交互的 AI 聊天助手。

## 核心目标

1.  **实现与本地 Ollama 模型的实时、可中止的流式通信。**
2.  **重构为经典的气泡式聊天界面，并实现完整、正确的交互功能。**
3.  **更新品牌 Logo 和模型列表。**
4.  **新增“数字人”页面，并集成Ollama AI交互与高级媒体控制功能。**

## 功能实现步骤

### 第一阶段：基础设置与品牌更新 (已完成)
*   已更新 Logo 和模型列表。

### 第二阶段：后端通信 (已完成)
*   已创建支持中止操作的 Ollama 服务。

### 第三阶段：UI 与交互实现 (已完成)
*   已实现消息内联编辑、中止生成和复制功能。

### 第四阶段：核心逻辑实现 (已完成)
*   已重构 `Home.vue` 以支持编辑、重新生成和中止逻辑。

### 第五阶段：新功能开发 (已完成)

1.  **创建“数字人”页面 (已完成)**:
    *   已在 `public/` 目录下创建 `js/` 文件夹，并放入 `jquery-3.6.0.min.js`, `client.js`, `sockjs.min.js`。
    *   已基于 `@webrtcapi.html` 的结构，在 `src/views/` 下创建 `DigitalHuman.vue` 页面组件。
    *   该组件已包含返回聊天页面的链接，并能动态加载和执行所需的JS脚本。
    *   已在 `src/router/index.js` 中为 `/digital-human` 添加新路由。
    *   已更新 `src/components/UserProfileMenu.vue`，将“数字人”链接指向新路由。
2.  **实现AI回复计时器 (已完成)**:
    *   **数据层**: 已在 `src/store/index.js` 的消息对象中添加 `duration` 属性，用于存储耗时。
    *   **逻辑层**: 已修改 `src/views/Home.vue` 的 `handleSendMessage` 方法，在请求开始时记录时间戳，在请求完成时计算总耗时并存入消息对象。
    *   **表现层**: 已修改 `src/components/MessageItem.vue`，在AI消息的操作栏中显示计时结果（例如 “2.5s”），并调整样式使其美观。

### 第六阶段：数字人页面功能增强与UI微调 (已完成)
1.  **集成Ollama AI**: 
    *   已在 `DigitalHuman.vue` 中集成 `ollama.js` 服务。
    *   用户输入现在会先发送给 `gemma3:1b` 模型进行处理。
    *   Ollama返回的完整响应文本再被发送给数字人进行播报。
2.  **增强媒体控制**:
    *   已为视频播放器添加全屏播放控件。
    *   已为页面添加了垂直滚动条，以适应不同尺寸的屏幕和内容。
3.  **UI样式微调**:
    *   已将 `TermsAndPolicies.vue` 中的返回链接向左移动。
    *   已修复 `SubscriptionSettings.vue` 中“管理订阅”按钮与关闭按钮重叠的问题。
